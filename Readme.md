# Testy Jednostkowe

Kilka przykładowych testów jednostkowych napisanych w JUnit dla funkcji wykonujących operacje na wartościach liczbowych i tekstowych.
Wykorzystano najpopularniejsze asercje m.in.
<ul>
  <li>assertEquals,</li>
  <li>assertNotNull,</li>
  <li>assertTrue,</li>
  <li>assertNotSame.</li>
</ul>
